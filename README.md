## M&O Accountans Demo.

# MO Accountants demo front-end.

### Getting started

#### Installing dependencies

```ts
  $ npm install
  $ yarn install
```

#### Configuration
Set backend URL in file `app/backend/common_api.ts`

```ts
const COMMON_API_URL = "http://127.0.0.1:3000/"
```

#### Start front-end service

#### In dev watch mode:
```
$ npm run dev
```
